# Grupi-5-FSHK-VITI-2
Ky repository përmban një mini-projekt të realizuar për kursin Modelime në Fizikë, i cili synon të prezantojë konceptet bazë të programimit shkencor duke përdorur Python. Projekti demonstron përdorimin e llogaritjeve numerike dhe vizualizimit të të dhënave me ndihmën e bibliotekave NumPy dhe Matplotlib.
